## Лабораторная работа №1: Элементы теории рекурсивных функций

**Вариант 21:** `f(x₁, x₂) = x₁ + x₂ - 1`

- Преобразование арифметической функции в рекурсивную

### Файлы:
- `lab1.py` — программа с двумя реализациями функции
- `README.md` — полный отчет по работе

### Запуск:
```bash
cd lab1
python3 lab1.py
```

### Ключевые результаты:
- Функция представлена через композицию базовых операторов: successor, predecessor, add
- Обе реализации дают одинаковый результат
- Пример: f(17, 19) = 35 ✓

## Лабораторная работа №2: Машина Тьюринга

**Вариант 21:** `f(x₁, x₂) = x₁ + x₂ - 1`

### Содержание:
- Построение программы для машины Тьюринга
- Симулятор МТ на Python (замена Algo2000 для macOS)
- Числа в унарной системе счисления

### Файлы:
- `turing_machine.py` — симулятор МТ
- `alphabet.txt` — внешний алфавит: `1 * λ`
- `program.txt` — программа МТ (6 команд)
- `tape.txt` — входная лента: `111*11`
- `output.txt` — трассировка выполнения
- `README.md` — полный отчет

### Запуск:
```bash
cd lab2
python3 turing_machine.py
```

### Ключевые результаты:
- Алгоритм: замена `*` на `1`, удаление двух последних единиц
- Состояния: Q = {q₀, q₁, q₂, q₃, qz}
- Пример: `111*11` → `1111` (3 + 2 - 1 = 4) ✓
- Пример: `11111*1111` → `11111111` (5 + 4 - 1 = 8) ✓

## Лабораторная работа №4: Конечные автоматы

**Вариант 7:** `(n|<b|d>)(h|k)<z|m>c`

### Содержание:
- Построение конечного автомата Мили
- Три способа задания КНА:
  - Матрицы переходов и выходов
  - Граф переходов (диаграмма)
  - Автоматная матрица
- Распознавание регулярных языков

### Файлы:
- `mealy_automaton.py` — симулятор КНА Мили
- `README.md` — полный отчет

### Запуск:
```bash
cd lab4
python3 mealy_automaton.py
```

### Ключевые результаты:
- Состояния: Q = {q₀, q₁, q₂, q₃, qf, qe}
- Входной алфавит: X = {n, b, d, h, k, z, m, c}
- Выходной алфавит: U = {0, 1}
- Допустимые слова: `nhc`, `bhzc`, `dkmc`, `hc`, `kzc` и др.
- Протестировано 36 слов: 24 допустимых + 12 недопустимых ✓
